# 📚 ElimuPro TZ

ElimuPro TZ ni platform ya kisasa ya elimu Tanzania inayosaidia wanafunzi na walimu kufanya maandalizi ya mitihani ya NECTA kwa urahisi zaidi.

## 🚀 Features Kuu

### 🎓 Kwa Wanafunzi
- Notes za masomo yote (Form 1–6 & Primary)
- Past Papers + Solutions (2000–2026)
- Daily Quiz system yenye feedback ya haraka
- Progress tracking na NECTA prediction
- Offline mode (kusoma bila internet)
- Video tutorials kwa topics ngumu

### 👨‍🏫 Kwa Walimu
- Schemes of Work & Lesson Plans (TIE format)
- Class management dashboard
- Auto-generate tests na reports
- Student performance analytics
- Offline mode kwa shule za vijijini
